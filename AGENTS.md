# AGENTS.md

This repository is skill-focused.

## Scope

Prioritize work inside:
- `skills/`
- `optional-skills/`
- `frameworks/`
- `docs/`

## Rules

- Keep `SKILL.md` concise and trigger-oriented.
- Put detailed material in `references/`.
- Put deterministic helper code in `scripts/`.
- Keep shared implementation in `frameworks/` only when needed.
- Do not reintroduce unrelated runtime, gateway, web, or test trees into this repo.
- When cleaning the repo, preserve files that are legitimate parts of a skill package, including `references/`, `scripts/`, `templates/`, and `assets/`.
